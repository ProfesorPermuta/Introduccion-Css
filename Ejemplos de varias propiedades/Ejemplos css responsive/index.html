Ejemplos Resposive
            La palabra responsive es un concepto clave en css. Este apartado es una colección de gestión responsive de css. 

            Responsive es un termino que se usa para hablar de la adaptavilidad de las páginas web a diferentes tamaños de pantalla. Ya sea modificando el contenido que se ve en la pantalla usando las sentencia @media o permitiendo que los elementos crezcan, reduzcan o reordenen en funcion del tamaño de pantalla.
    
            Os aconsejo visitar los siguientes link además de ver los ejemplos mostrados para entender en profundidad el concepto de responsive.
