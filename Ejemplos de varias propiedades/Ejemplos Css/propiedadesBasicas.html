border: Sirve para crear un borde al elemento. Aunque box shadow no es exactamente un borde, me parece que cuadra en esta sección
Margin es el espacio entre un elemento y el siguiente
Padding es el espacio entre el contenido de un elemento y su borde
Se puede aplicar un -top a ambos para indicar si el margin o el padding es superior, inferior, etc
                
                    #margin{
                        margin: 10px; /*Espacio entre un elemento y el siguiente*/
                    }
                    
                    #padding{
                        padding: 10px; /*Espacio entre el borde de un elemento y su contenido*/
                    }
                    
                    #marginTop{
                        margin-top: 10px; /*Se puede aplicar en margin y padding en las 4 direcciones*/
                    }
Margin y padding: Son usados para crear espacio entre elementos
