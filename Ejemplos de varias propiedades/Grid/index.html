Ejemplos básicos de grid usando Css
            Display grid esta cada vez obteniendo más relevancia en su uso sustituyendo a display flex en muchos aspectos del diseño de páginas web. Esto se debe a que es más fácil visualizar los espacios reservados por la página para cada sección sin desde el principio del desarrollo. 
            Tambien es más facil gestionar diferentes tamaños de pantalla al poder simplemente ocultar una columna o fila por completo siendo el resultado final lo esperado.

            Aunque normalmente hablamos de responsive cuando usamos flex, grid tambien es una propiedad responsive, por lo que su uso está más aconsejado que otros posicionamientos como position o float.
