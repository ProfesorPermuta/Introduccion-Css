Ejemplos básicos de Posicionamiento usando Css
Estoy centrado
                 
.elementoCentrado{
    /*Propiedades necesarias para centrar el contenido*/
    width: 70%;
    margin: auto; /*El margin auto sirve para que margen se calcule dejando el mismo espacio a la izquierda que a la derecha*/
}
Centrar elemento: Existen varias maneras de centrar elementos dentro de su contenedor. En este ejemplo usaremos una convinación de margin y width. Tendremos un contenedor principal al que le daremos un color para que sea más facil de visualizar el ejemplo.
Estoy centrado
                     
#contenedorPrincipalFlex{
    /*Propiedades necesarias para centrar el contenido*/
    width: 100%;
    display: flex;
    justify-content: center;
}
Centrar elemento: Existen varias maneras de centrar elementos dentro de su contenedor. En este ejemplo usaremos flex y justify-content. Fijaos que a diferencia del ejemplo anterior esta vez es el contenedor el que decide como se visualizan los elementos.
top:20px left:15px
                 
#relativePositionItem {
    position: relative;
}

.absolutePositionElement {

    /*No se recomienda usar el position absolute en general a la hora de trabajar en posicionamiento. 
    Si aún así se utiliza, porfavor que sea siempre como hijo de un position relative*/
    position: absolute;
    top: 20px;
    left: 15px;
}
Position absolute: Sirve para posicionar elementos de manera absoluta al primer progenitor que contenga la propiedad relative. Su uso está desaconsejado, pero en caso de usarlo que el elemento absolute siempre esté dentro de un elemento relative
top:20px left:15px
                 
#gridContainer {
    display: grid;
    /*Indicamos el ancho de las columnas que queremos crear en nuestro grid
    si lo marcamos como auto el grid se ajustará para ocupar el espacio necesario dependiendo del contenido*/
    grid-template-columns: 33% 33% auto;
    grid-template-rows: auto auto;

}

.gridElement {

    /*Ahora podemos tratar las celdas como si fuesen elementos normales, solo tenemos que indicar la posicion que ocupan en el grid*/
    grid-column: 1;
    grid-row: 1;
    margin-left: 20px;
    margin-top: 15px;
}
Display grid: Nos permite gestionar el contenedor como si se tratase de un grid. Existen multitud de propiedades asociados al display grid. Revisar el link W3Schools si os interesa profundizar más.
